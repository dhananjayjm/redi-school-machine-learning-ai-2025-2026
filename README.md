# ReDI School Machine Learning / AI
### Volunteer Instructor Resources | Fall 2025 & Spring 2026

This repository contains the teaching materials, practical examples, assignments, and supporting resources that I developed while volunteering as a **Machine Learning / AI Instructor** at the **ReDI School of Digital Integration, Berlin** during the **Fall 2025** and **Spring 2026** semesters.

## About

ReDI School of Digital Integration is a non-profit organization that provides free technology education to newcomers and underrepresented communities, helping them build digital skills and access employment opportunities.

As a volunteer instructor, I contributed to the 14 week **Machine Learning / AI** course delivered in English.

## Teaching Periods

| Semester | Duration |
|----------|----------|
| Fall 2025 | 08 September 2025 – 12 December 2025 |
| Spring 2026 | 16 March 2026 – 19 June 2026 |

## Responsibilities

During both semesters, I contributed to:

- Teaching Machine Learning and AI concepts in English
- Preparing lectures and learning, homework materials
- Designing practical programming exercises
- Creating Python examples and notebooks
- Assessing homework and final projects
- Supporting students during practical sessions
- Collaborating with fellow volunteer instructors
- Coordinating with the ReDI School teaching team to improve the learning experience

## Course Topics

The course focused on practical Machine Learning using Python, including topics such as:

- Python for Data Science
- NumPy
- Pandas
- Data Visualization
- Data Preprocessing
- Supervised Learning
- Unsupervised Learning
- Model Evaluation
- Feature Engineering
- Scikit-learn
- Machine Learning Workflows
- AI Fundamentals
- Final Machine Learning Projects
