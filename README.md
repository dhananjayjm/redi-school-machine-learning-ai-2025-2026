## About

This repository documents my volunteer teaching experience as a **Machine Learning / AI Instructor** at **ReDI School of Digital Integration, Berlin** during the **Fall 2025** and **Spring 2026** semesters.

It contains lecture materials, practical Python notebooks, assignments, classroom resources, and project guidance developed to help students learn Machine Learning and Artificial Intelligence through hands-on programming.

ReDI School is a non-profit organization that provides free technology education to newcomers and underrepresented communities, helping learners develop digital skills and improve access to employment opportunities.
